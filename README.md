# base
* 一个简单的CPU版本的梯度下降部分的代码
* 可以单独编译运行
* 代码习惯：三角形用逆时针表示，默认数组从0开始，非法使用负数或者-1或者nullptr表示，区间表示上，左闭右开和左闭右闭混合使用。
# 编译运行
* 在ubuntu环境下编译运行
* 需要安装glfw和libpng库
* 
* 在终端中如下输入
```
cd base
sudo apt install libpng++-dev
sudo apt install libglfw3 libglfw3-dev
make
./style pic/dong.png
```
